atualizarPP(mes, erros, dataValor) → {void}

Utilizada pelo robô de consulta na caixa Atualiza a aba "3PP" em um arquivo externo com base nos resultados de uma aba local "resultado pp". A função extrai o ID da planilha externa de uma fórmula IMPORTRANGE, mapeia os dados de "resultado pp" e os usa para atualizar as colunas "Contas", "Saldo Total", o mês específico, "Status" (marcando erros) e "Consulta" na planilha "3PP". Após a atualização, a aba "resultado pp" é limpa.
Parameters:
Name Type Description
mes string O nome do mês (ex: "JAN", "FEV") cuja coluna de valor pago deve ser atualizada na planilha "3PP".
erros Array.<string> Um array de IDs de processo (limpos de pontuações) que devem ser marcados com "Erro" na coluna "Status" da planilha "3PP".
dataValor string | Date A data da consulta ou atualização a ser registrada na coluna "Consulta" da planilha "3PP". Pode ser uma string de data ou um objeto Date.
